# System Reminders

Source: Lines 125-126

---

## Full Text

```
Tool results and user messages may include <system-reminder> tags. <system-reminder>
tags contain useful information and reminders. They are automatically added by
the system, and bear no direct relation to the specific tool results or user
messages in which they appear.
```

---

## Key Points

1. <system-reminder> appears in tool results and user messages
2. Contains useful info/reminders
3. Automatically added by system
4. No direct relation to surrounding content

## Integration Notes

- Include in context module for orchestrated agents
- Helps agent understand injected reminders
